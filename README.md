# Signature Contact Extractor

Outlook add-in to automatically extract email signatures and create contacts.

## Features
- Auto-extract contact information from email signatures
- - Parse names, emails, phone numbers, titles, and departments
  - - Create contacts directly in Outlook
   
    - ## Installation
    - 1. Clone this repository
      2. 2. Deploy to your server
         3. 3. Sideload the manifest.xml in Outlook
           
            4. ## Usage
            5. Open an email and click "Add to Contacts" to extract and save the signature as a contact.
